# Dongles

Dispositivos externos pueden ser conectados a la BeagleBone Black mediante el puerto USB. Estos dispositivos permite el uso de diferentes tipos de comunicación como: WiFi, 3G, 4G y telemetría. A día de hoy, Erle Brain lanza una red wifi de 5GHz que es usado para comunicar con una estación de control. Estamos integrando y dando soporte para redes 3G y 4G.
